# Signal
This is a signal class I built for my utillities in my games feel free to use! 

The point of this class is to mimik roblox's API of current signal class. This signal class currently implements all functions of a normal RBXSignal with type safety included. The one method not implemented is the ConnectParellel(I do not really know how I would even do this...). Other than that this Signal class is very easy to use by using function prototypes so you do not get lost in your own code and auto correct the whole time!
